# Предварительные требования (Windows)

Установите:

1) **GTA V** (лицензия)
2) **FiveM клиент**: https://fivem.net/
3) **Git**: https://git-scm.com/downloads
4) **Docker Desktop**: https://www.docker.com/products/docker-desktop/
5) **7-Zip**: https://www.7-zip.org/

Дополнительно:
- PowerShell 5+ (обычно есть в Windows 10/11)
- Доступ в интернет для скачивания FXServer и ресурсов
